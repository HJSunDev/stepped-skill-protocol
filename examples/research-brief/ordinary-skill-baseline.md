# Ordinary Skill Baseline

This baseline is used for M1 eval against the Stepped Skill sample.

A comparable ordinary Skill would place the full workflow in one activated `SKILL.md`:

1. Clarify topic and scope.
2. Collect relevant sources and facts.
3. Synthesize major findings.
4. Write final brief with source notes and caveats.

Expected weakness compared with SSP:

- collection, synthesis, and finalization instructions are all visible at once;
- the model may prematurely draft final prose during collection;
- intermediate state may be implicit instead of captured in handoff;
- later output requirements can distract from current-step source work.

Eval should compare this baseline against the SSP package on the same tasks, same model, same tools, and same source access.
